# ihistory

A fast, fuzzy shell history search.

![demo](demo/demo.gif)

## Install

```bash
brew install jazzjackrabbit/tap/ihistory
ihistory --init
```

## Usage

```bash
ih           	# (or ihistory) launch 
ih git       	# search with initial query
ih "git tag"	# search with initial query 
```
